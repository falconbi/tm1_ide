Core Features

Rules Editor

Monaco-based editor with TM1 syntax highlighting, live validation via CheckRules, code formatter, regions, snippets, and cell calculation trace.

TI Process Editor

Full TurboIntegrator editor with 4-section tabs, parameter editor, datasource config, run with return, step debugger, and error log viewer.

Dimension Editor

Hierarchy tree with element and edge CRUD, attribute grid, bulk import, and element rename tracking.

View Editor

Native and MDX view builder with live cell grid, writeback support, and subset assignment per axis.

Subset Editor

MDX code view and visual tree subset builder with static and MDX save, preview execution, and guided filter builder.

Chore Editor

Schedule editor with step management, activate/deactivate, and manual execute with live status feedback.

SQL Editor

Multi-driver external SQL connections (MSSQL, PostgreSQL, MySQL, SQLite) with schema browser, saved queries, and post-to-TI.

Global Search

Cross-object search across all rules and TI code on the server. Find any function, cube reference, or variable across the entire model.

Deploy Pipeline

Change set tracking, baseline diff, package builder, risk analyser, and server-to-server deployment — with full audit trail.

Dev to Test in four steps

The built-in deploy pipeline tracks every change you make, compares against a baseline snapshot, packages only what changed, checks for risks, and deploys to any registered target server.

1 Diff — compare session changes against baseline
2 Package — fetch live objects and write to disk
3 Risk — syntax check, dependency check, chore conflict
4 Deploy — push to target server with dry-run option
