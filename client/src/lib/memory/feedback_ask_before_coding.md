Always ask and confirm the plan before writing any code. Do not start implementing after a discussion — wait for explicit go-ahead from the user.

**Why:** User explicitly asked for this after I started writing code mid-conversation without checking first.

**How to apply:** After proposing a plan or list of changes, stop and wait for confirmation. Only proceed when the user says yes/go ahead/proceed or similar.
